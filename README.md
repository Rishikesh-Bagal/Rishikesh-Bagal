## Featured Projects

### [SkillSwap](https://github.com/Rishikesh-Bagal/SkillSwap-Capstone)
Peer-to-peer skill exchange platform with real-time presence, live chat, an AI quiz system powered by Gemini, and a full Learn Hub. Built with Firebase Realtime Database and Firestore for live sync across sessions.

**Stack:** `React` `TypeScript` `Firebase` `Gemini AI` `Tailwind CSS`

---

### [Blogify API](https://github.com/Rishikesh-Bagal/Blogify_API)
Production-grade blogging backend with JWT auth, Stripe payments, Cloudinary media uploads, and Docker containerisation. Deployed on Render.

**Stack:** `Node.js` `Express` `MongoDB` `JWT` `Stripe` `Docker`

---

### [IKS Study Hub](https://github.com/Rishikesh-Bagal/IKS-Study-Hub)
AI-powered learning platform for Indian Knowledge Systems with Gemini AI integration. Built for a real academic assignment with a production-quality frontend.

**Stack:** `React` `TypeScript` `Firebase` `Gemini AI`

---

### [Creators Platform](https://github.com/Rishikesh-Bagal/creators-platform)
Full-stack blog application with user auth, CRUD operations, pagination, and Docker support. A clean MERN implementation with JWT-protected routes.

**Stack:** `MongoDB` `Express` `React` `Node.js` `JWT` `Docker`

---

### [Document Guide](https://github.com/Rishikesh-Bagal/Document_maker_guide)
Multilingual government document guidance platform built during a hackathon. Helps users in India navigate complex government paperwork in their native language.

**Stack:** `HTML` `CSS` `JavaScript`
