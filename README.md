## What I've built

### [SkillSwap](https://github.com/Rishikesh-Bagal/SkillSwap-Capstone)
> P2P skill exchange with real-time presence, live chat, AI quiz engine (Gemini), and a Learn Hub — all synced via Firebase Realtime Database and Firestore.

`React` `TypeScript` `Firebase` `Gemini AI` `Tailwind CSS`

---

### [Blogify API](https://github.com/Rishikesh-Bagal/Blogify_API)
> Production backend with JWT auth, Stripe payments, Cloudinary uploads, and Docker. Deployed live on Render.

`Node.js` `Express` `MongoDB` `JWT` `Stripe` `Cloudinary` `Docker`

---

### [IKS Study Hub](https://github.com/Rishikesh-Bagal/IKS-Study-Hub)
> AI-powered learning platform for Indian Knowledge Systems. Gemini AI integration with a production-quality React frontend.

`React` `TypeScript` `Firebase` `Gemini AI`

---

### [Creators Platform](https://github.com/Rishikesh-Bagal/creators-platform)
> Full-stack blog app — auth, CRUD, pagination, Docker. Clean MERN with JWT-protected routes.

`MongoDB` `Express` `React` `Node.js` `JWT` `Docker`

---

### [Document Guide](https://github.com/Rishikesh-Bagal/Document_maker_guide)

`HTML` `CSS` `JavaScript`
